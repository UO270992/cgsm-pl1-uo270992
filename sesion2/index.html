Descripción de la práctica

Durante esta práctica aprendimos sobre el uso de algunos elementos básicos de la librería WEBGL. Entre estos destacamos los planos y la cámara para visualizar dichos objetos. Los objetos pueden estar predefinidos con ciertas geometrías o pueden crearse de 0 como veremos más adelante. Además aprendimos un poco sobre los distintos materiales que pueden formar las geometrías, así como el uso de algunas luces para verlos.
