Memoria

Descripción de la práctica

En esta sesión seguiremos trabajando en aspectos avanzados de three.js. Concretamente, utilizaremos sonidos 3D, de forma que el usuario perciba cómo el sonido cambia cuando se desplaza por la escena. Además, aprenderemos a seleccionar objetos en la escena y detectar colisiones con ellos.

Ejercicio 1

Añadimos audio a ambas cajas para aprender a utilizar sonido posicional, por comodidad no se reproduce el audio de manera automática si no que se espera interacción del usuario

Ejercicio 2

En esta sección implementamos selección de objetos mediante trazado de rayos. Podemos ver en la consola como aparece el nombre de los objetos cuando pasamos el cursos sobre ellos.

Ejercicio 3

Con el selector de objetos implementado podemos hacer que las cajas se activen cuando pulsamos el espacio con el ratón encima de ellas. También modificamos la textura para tener un efecto visual a la hora de reproducir el sonido

Ejercicio 4

Por último implementamos un mecanismo para detectar colisiones, trazando rayos en todas las direcciones desde la cámara. Se puede comprobar que no es posible atravesar los objetos en este escenario.

Dificultades encontradas

Considero que la mayor dificultad es comprender cómo funcionan los audios en WEBGL, y que estos tienen que aplicarse al objeto para obtener audio posicional. También que hay que ser precavido cuando se utiliza trazado de rayos ya que es necesario definir bien qué objetos pueden ser intersecados.
